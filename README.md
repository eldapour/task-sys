# Tasks Management System
***
## Installation

### requirements :
* php >= 8.0

```
composer install
```
### command

* migration 
```
php artisan migrate:fresh --seed
```
* start server
```
php artisan serve
```
* seed tasks by queue
```
php artisan queue:work --queue=task_seeder
```
* run all schedule
```
php artisan schedule:work
```
* when read all notify for admin
```
php artisan queue:work --queue=read_notify
```
